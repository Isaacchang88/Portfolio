SOIL — Full Stack Web Application

A multi-user social commerce platform built for RMIT's Full Stack Programming course, taking a React front-end prototype to a complete full-stack application with a Node/Express/GraphQL backend, Cloud MySQL database, and a separate admin dashboard.

‹ ›

Project Context

SOIL began as a front-end prototype of RMIT's Full Stack Programming. The brief was to take that prototype and build it out into a genuine full-stack, multi-user web application — connecting it to a real Cloud MySQL database via a custom REST API, while also designing and building a completely separate GraphQL-powered admin dashboard for moderation and product management.

Built in a group, the project required strict adherence to a fixed technology stack — functional React, Node.js/Express with Sequelize ORM for the REST layer, and GraphQL with subscriptions for the real-time admin dashboard — along with hashed password storage, a normalised relational schema, and full version control via GitHub.

Full Stack · Group Project

SOIL is a social shopping platform where users can sign up, build a profile, follow and unfollow other users, browse standard and special products, leave star-rated reviews, and check out through a full shopping cart and payment flow. All data operations run through a custom REST API built with Express and Sequelize ORM against a Cloud MySQL database, with passwords stored using bcrypt hashing rather than insecure algorithms like MD5.

A second, entirely separate Admin Dashboard application was built to manage the platform — using GraphQL (with subscriptions for real-time updates) instead of REST, as required by the brief. Admins can block/unblock users, manage the product catalogue, moderate inappropriate reviews, and view live engagement metrics through charts that update as users interact with the site.

Auth & Profiles
Sign-up/sign-in with hashed passwords, unique email enforcement, dual-layer form validation (React + API), and editable user profiles fetched live from the database.
Products & Reviews
Standard and special product listings pulled via the API, with star-rated reviews (1-5), edit/delete on your own reviews, and threaded reply support for postgraduate scope.
Cart & Checkout
Full shopping cart for standard and special products, all state and operations backed by the API, plus a dedicated payment flow to complete a purchase.
GraphQL Admin Dashboard
Separate React app and GraphQL API for user/product management, review moderation, and a live-updating metrics view powered by GraphQL subscriptions.
