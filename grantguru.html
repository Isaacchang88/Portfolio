GrantGuru Partner Portal Admin

A full-featured admin portal for GrantGuru's partner organisations - rebuilt from React to Astro — enabling partners to independently manage their users, portal content, and branding.

‹ ›

Project Context

GrantGuru is an Australian grant-discovery platform serving local councils, community organisations, and businesses. Partners - such as councils in regional WA - white-label a GrantGuru portal for their community. This admin portal is how those partners control everything their end-users see: the home page imagery, welcome copy, navigation tiles, and the list of registered users.

The original portal was built in React and had not been updated since late 2023. The brief was to faithfully rebuild it in Astro while applying an updated design based on GrantGuru's style guide (Figma), improving the UX of several flows, and laying groundwork for new features like team management and bulk user actions.

Full Stack · Web Application

Contracted by GrantGuru to rebuild their existing React-based partner admin portal using Astro 5, Tailwind CSS 4, and TypeScript. The portal lets council, member, and corporate partners manage their own users, brand settings, home page layouts, and media assets - all without touching the codebase. Login credentials are encrypted with Crypto-js before transmission and the MFA flow was simplified from a six-box code entry to a single input field.

User Management
Bulk actions, per-row context menus, CSV export, invite flow with email check, and role management (Admin / Team Lead).
Partner Content
Home page layout picker (4 templates), banner/tile/welcome editors, social media link manager, and logo + colour branding.
Image Library
Drag-and-drop or URL fetch uploads to AWS S3, with automatic resizing to per-slot dimension limits (banner, tile, logo, welcome).
Auth & Security
Encrypted credentials via Crypto-js, MFA with single-field verification code, and auto-login from the main GrantGuru portal.
Astro 5 Tailwind CSS 4 TypeScript Crypto-js Node.js AWS S3 REST API
